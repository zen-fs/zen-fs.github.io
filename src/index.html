Welcome to the ZenFS Docs!

You can find the docs for specific @zenfs modules below:

core : The core, includes some backends

dom : Backends which use DOM APIs (i.e. browsers)

archives : Backends for archive files like .zip and .iso

cloud : Backends for cloud providers, like Google Drive, Dropbox and AWS S3.

emscripten : Backend for use with Emscripten (Work in Progress)
